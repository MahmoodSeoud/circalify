See It In Action

Create a timeline in 3 lines of code. Customize with your data.

const timeline = new CircularTimeline('#timeline', {
startYear: 2025,
rings: [{ type: 'calendar' }, { type: 'data' }]
});
timeline.setData([
{ label: 'Launch', startDate: '2025-03-15' }
], 'Events');
