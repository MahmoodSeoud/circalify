Multiple DataRings

This demo shows 3 DataRings with separate event data:

Core Events (Inner)

6 events - Board meetings and reviews

wheel.setData(coreEvents, "Core Events")

Projects (Middle)

5 events - Project milestones and launches

wheel.setData(projectEvents, "Projects")

Team Events (Outer)

7 events - Team activities and workshops

wheel.setData(teamEvents, "Team Events")

Note: Each DataRing maintains its own independent event data!
